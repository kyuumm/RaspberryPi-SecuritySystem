# RaspberryPi-SecuritySystem
## 作品背景
当前家庭及办公环境对安防智能化的需求持续增长，传统方案多依赖单一传感手段，易受环境干扰，且缺乏身份甄别能力，导致误报频发、响应滞后。本作品以树莓派为核心，融合磁控开关、光遮断检测与视觉识别技术，构建了一套分级触发的智能安防原型系统，力求在低成本下提升监测准确性与交互直观性。
## 使用到的组件
- 树莓派（Raspberry Pi）：主控平台，统筹传感数据采集、人脸推理与外围驱动。
- 干簧管传感器：检测磁铁靠近与否，用作系统休眠/唤醒的物理开关。
- 光电传感器（光隔离）：探测通道遮挡，作为人员经过的第一道触发信号。
- 双色LED（红/绿）：状态指示，红色为待机，绿色为告警。
- 激光模块：告警时点亮，强化警示效果。
- LCD1602液晶屏（I2C）：显示当前工作模式与提示信息。
- USB摄像头：实时抓拍画面，供人脸检测算法分析。
## 重要模块介绍
- 唤醒与休眠控制：干簧管作为总开关，无磁时系统彻底待机，有磁时激活全部功能，既节能又便于手动布防/撤防。
- 双重触发逻辑：光电传感器负责物理层遮挡检测，摄像头则执行语义层人脸确认——两者“与”条件成立才触发最终报警，显著降低飞虫、光影变化等造成的误报。
- 人机交互输出：激光灯与双色LED提供视觉强提醒，LCD屏同步显示“System Closed”“Monitoring”“Alarm!”等状态，便于现场人员快速掌握系统态势。
## 系统功能介绍
上电后，若干簧管未感应到磁铁，所有外设关闭，LCD显示系统关闭。

当磁铁靠近，系统进入监控模式：此时若光电传感器无遮挡，LED保持红色，表示安全待机；一旦有物体阻断光路，摄像头立即抓拍并执行人脸检测。若成功检出人脸，系统判定为有效入侵，激光模块点亮、LED切换为绿色、LCD显示报警文本；若未检到人脸，仅记录遮挡事件但不触发告警，继续保持监测状态。整个过程循环运作，实现“磁控唤醒—物理触发—视觉确认—分级响应”的闭环逻辑。
## 应用场景
本作品可灵活适配多种中小型安防需求，例如：家庭门窗状态监测与陌生人闯入提醒；公司访客接待区的自动感知门铃；实验室或仓库的限定区域进出管控；智能展柜的靠近告警；校园安防教学中的综合传感实验平台等。
